# Simulasi Pengaturan Kipas Angin Menggunakan Metode Fuzzy

## Deskripsi

Program ini dibuat untuk mensimulasikan pengaturan kecepatan kipas angin menggunakan metode Fuzzy Logic berdasarkan suhu dan kelembapan.

## Tools

* Python
* NumPy
* scikit-fuzzy

## Cara Menjalankan

1. Install library:
   pip install numpy scikit-fuzzy

2. Jalankan program:
   python main.py

3. Masukkan nilai suhu dan kelembapan

## Output

Program menghasilkan kecepatan kipas dalam bentuk nilai dan kategori:

* Lambat
* Sedang
* Cepat
